# My Keep 📝

A professional, clean personal notes app built with Flutter.

## Features
- ✅ Clean minimal UI — warm white & yellow theme
- ✅ Dark / Light mode (right drawer toggle)
- ✅ PIN lock for notes (single PIN, all locked notes)
- ✅ Pin notes to top
- ✅ Color-coded note cards (7 colors)
- ✅ Search notes
- ✅ Categories / Labels
- ✅ Trash with restore (30-day retention)
- ✅ 100% offline — SQLite local storage
- ✅ No internet required

## How to Get the APK (GitHub Actions)

### Step 1 — Create GitHub Repository
1. Go to [github.com](https://github.com) and sign in
2. Click **New repository**
3. Name it `my-keep`
4. Set to **Public**
5. Click **Create repository**

### Step 2 — Upload Files
Upload all files maintaining this folder structure:

```
my-keep/
├── .github/
│   └── workflows/
│       └── build.yml
├── lib/
│   ├── main.dart
│   ├── models/
│   │   └── note_model.dart
│   ├── helpers/
│   │   └── database_helper.dart
│   ├── providers/
│   │   └── theme_provider.dart
│   ├── screens/
│   │   ├── home_screen.dart
│   │   ├── note_editor.dart
│   │   ├── lock_screen.dart
│   │   ├── settings_screen.dart
│   │   └── trash_screen.dart
│   └── widgets/
│       ├── note_card.dart
│       └── drawer_widget.dart
├── assets/
│   ├── fonts/    ← (auto-downloaded by GitHub Actions)
│   └── icons/    ← (auto-generated by GitHub Actions)
├── android/
│   └── (standard Flutter android folder)
├── pubspec.yaml
└── README.md
```

### Step 3 — Wait for Build
1. Go to your repo → **Actions** tab
2. You'll see **Build APK** workflow running
3. Wait ~15-20 minutes

### Step 4 — Download APK
**Option A — From Releases:**
- Go to repo → **Releases** → Download `MyKeep-v1.0.0.apk`

**Option B — From Actions:**
- Go to **Actions** → click the latest run → scroll down → **Artifacts** → Download `MyKeep-APK`

### Step 5 — Install on Phone
1. Transfer APK to your Android phone
2. Open the APK file
3. If prompted, enable **"Install from unknown sources"**
4. Install and enjoy!

## Android Folder Setup

You need a standard Flutter android folder. The easiest way:

1. Install Flutter locally: https://flutter.dev/docs/get-started/install
2. Run: `flutter create my_keep`
3. Copy the generated `android/` folder into your project
4. Upload it to GitHub

Or use the **flutter-action** in GitHub Actions which handles this automatically.

## Lock System

- **First time locking a note:** You'll be prompted to set a 4-digit PIN
- **Enter + Confirm** the PIN to set it
- **Same PIN** is used for all locked notes
- **Delete a locked note:** Requires PIN verification
- **Change PIN:** Settings → Change PIN
- **Remove PIN:** Settings → Remove PIN (unlocks all notes)
